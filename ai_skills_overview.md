# AI Skills Overview – ClearAid

ClearAid uses a reusable five-skill architecture to support different stages of the user journey.

## 1. eligibility_precheck_router
Narrows likely benefit pathways using minimal user context and avoids declaring official eligibility.

## 2. plain_language_explainer
Rewrites confusing application language into simple, reassuring, multilingual explanations.

## 3. document_checklist_builder
Generates a practical “what to prepare” checklist with alternatives when documents are missing.

## 4. next_step_advisor
Provides prioritized next actions based on urgency, preferred channel, and user barriers.

## 5. application_navigator
Guides the user through a structured application flow one step at a time with progress tracking and confirmation gating.

## Why this matters

This architecture turns ClearAid into a guided AI product, not just a static informational prototype.
